# One-line thesis

Learnable Token Dictionary + cross-attention enables global dependency modeling with linear complexity for image restoration.

## Problem / Gap

Self-attention in transformers has quadratic complexity → limited to local windows → limited receptive field.

## Method

- Token Dictionary: learnable `nn.Parameter(num_tokens, dim)` storing typical image structure priors
- ATD_CA: cross-attention between features and Token Dictionary → sim_atd as similarity scores
- AC_MSA: category attention using tk_id from argmax(sim_atd) to group similar features
- SW-MSA: shifted window attention for local processing
- Three-branch fusion: x = shortcut + x_atd + x_win + x_aca

## Key Results

SOTA on multiple SR benchmarks. ATD-light is lightweight variant. ATD-U for multi-scale restoration.

## Assumptions

- Token Dictionary can capture meaningful image priors through training
- Static dictionary is sufficient for all images/timesteps

## Limitations / Failure Modes

- Token Dictionary is STATIC — same for all timesteps/degradation levels
- No mechanism to adapt memory retrieval based on input conditions

## Reusable Ingredients

- Token Dictionary + ATD_CA as a memory mechanism
- AC_MSA category-based grouping
- The three-branch (TD + window + category) attention pattern

## Open Questions

- How to make Token Dictionary adaptive to timestep/degradation?
- Would timestep-conditioned TD improve flow matching SR?
